Wurzelstrang is a simple cms, created and maintained by Nikolaus Höfer.

  • Written in HTML, CSS, PHP and JavaScript
  • Using sqlite to store data, Slim Framework for the REST API, jQuery for the admin interface, Mozilla's Persona for accounting, CKEditor and KCFinder for content editing and Kube for Administration-Interface's CSS styling
  • Server-requirements: php5, sqlite, php-curl(recommended)
