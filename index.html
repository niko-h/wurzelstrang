About

    The idea behind Wurzelstrang is to give you a lightweight CMS at hand that does not get in your way of designing a website. It is designed for small websites with easy structures. The admin-interface is meant to be as easy as possible, so literally everybody can handle it. For now the interface is in german.

    Written in HTML, CSS, PHP and JavaScript, using sqlite to store data, Slim Framework for the REST API, jQuery for the admin interface, Mozilla's Persona for authorization, CKEditor and KCFinder for content editing and Kube for Administration-Interface's CSS styling.
    Little server-requirements: php5, sqlite, php-curl(recommended)
